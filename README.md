# ЗАРЯД · Табель учёта рабочего времени

Веб-приложение для учёта рабочего времени. Чистый Python stdlib — без сторонних зависимостей.

## Запуск

```bash
python app.py
```

Открыть в браузере: `http://localhost:9090`

## Конфигурация `.env`

```env
WEB_USERS=admin:пароль,boss:пароль2
WEB_PORT=9090
WEB_HOST=0.0.0.0
WORKTIME_DB=data/worktime.db
```

## Структура

```
app.py          — точка входа
config.py       — конфиг и загрузка .env
utils.py        — утилиты (время, периоды, расчёт часов)
sessions.py     — сессии в памяти
db/             — работа с SQLite
views/          — HTML-страницы
handler/        — HTTP-роутинг
```

## Возможности

- Дашборд с графиками и детализацией по периодам
- Массовая отметка прихода / ухода
- Добавление смен задним числом
- Профили работников с heatmap и историей
- Объекты и прикрепление работников к ним
- Личный кабинет работника (`/my`)
- Экспорт в CSV
- Два типа пользователей: `admin` и `worker`
